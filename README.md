# Robot Gladiators!

## 1. Description
Robot Gladiators is a game made using JavaScript that allows the user to name a character and fight robots.

## 2. Features
 * Name your robot
 * Battle enemy robots
 * Visit shop 
     * buy items
     * increase power
 * Get a total score at end game
